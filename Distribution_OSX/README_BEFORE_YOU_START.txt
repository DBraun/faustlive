
==================== BEFORE YOU START USING FAUSTLIVE ===================

-----Configure File Associations 

To ensure that your system is well configured for .dsp, .lib and .svg files, please configure DefaultDSP.dsp, DefaultDSP.lib and process.svg : 

1- Right Click on the file
2- Choose Read Information
3- Choose the default application with which the file will be opened
	(a text editor for .dsp and .lib ; 
		a browser for .svg)
4- Click "Change All"


-----Install JACK (or update it)

Double Click on Jack Installation package.
